01 — WEB APPLICATION

言の葉

Kotoba no Ha

Planning / UI Design / Worldview Construction

言葉が葉っぱのように舞い落ちる、動的な学習体験。
単語を静止した記号としてではなく、時間とともに流れる「現象」として捉えました。

ただ暗記するのではなく、落ちてくる言葉に反応（Reaction）するゲーム性を取り入れることで、学習への没入感を高めています。

UI / UX FOCUS
落下アニメーションによる時間の可視化
直感的な速度コントロール（重力の調整）
禅（Zen）の美意識に基づいたミニマルな配色

02 — MOOD LOGGING SITE

ひとりぼし

Hitoriboshi

Planning / UI Design / Worldview Construction

「ひとりぼっち」という言葉は、しばしば孤独というマイナスの意味で捉えられます。
しかし、「ひとりぼし（独り星）」と言い換えることで、それは自分だけの静かで大切な空間、あるいは夜空に光る一つの星というポジティブなイメージに変わります。

他人の目線を気にせず、自分の思考や想いを馳せることができる「内省のためのサンクチュアリ」として設計しました。

UI / UX FOCUS
没入感を高める星空のパーティクル表現
感情を惑星に見立てたカラー分類システム
自己との対話を促す「自分への引用（Retweet）」機能
